# Archive

This folder stores generated or historical files that are not part of the active application bundle.

- `generated/figma-imports`: old Figma-generated pages and image assets.
- `generated/figma-components`: old Figma helper components.
- `generated/shadcn-ui`: unused generated UI component library files.
- `generated/template-bootstrap`: unused template/bootstrap styling and guidelines from the original generated project.
- `generated/old-package-manager`: old pnpm lock/workspace files. The active project README uses npm.

Keeping these files here makes the active `src` tree easier to review without deleting previous generated work.
