# Prompt Library cho do an MiniStyle

## 1. Phan tich yeu cau

```text
Hay phan tich yeu cau do an website ban hang mini tich hop chatbot AI. Liet ke chuc nang bat buoc, du lieu can chuan bi, rui ro va tieu chi demo thanh cong.
```

## 2. Tao sitemap

```text
Tao sitemap cho website ban quan ao mini gom Trang chu, San pham, Chi tiet san pham, Gio hang/Thanh toan va chatbot tu van.
```

## 3. Thiet ke landing page

```text
De xuat bo cuc landing page cho shop quan ao MiniStyle. Phong cach hien dai, mau sac de nhin, co hero, danh muc noi bat, san pham ban chay va loi ich cua shop.
```

## 4. Viet du lieu san pham

```text
Tao 8 san pham mau cho shop quan ao sinh vien. Moi san pham co ten, gia, danh muc, mo ta ngan, doi tuong phu hop, size, mau sac va tag de chatbot tu van.
```

## 5. Viet React component

```text
Viet component ProductCard React cho san pham quan ao. Card can co anh, ten, gia, mo ta, badge, nut xem chi tiet va nut them vao gio hang.
```

## 6. Debug UI

```text
Kiem tra giao dien React/Tailwind cua toi va chi ra loi UX: chu qua dai, button khong ro, layout mobile loi, mau sac kho doc, spacing khong deu.
```

## 7. System prompt chatbot

```text
Viet system prompt cho chatbot tu van san pham cua shop quan ao. Chatbot chi duoc tra loi dua tren du lieu san pham va chinh sach shop, khong bia thong tin.
```

## 8. Tao FAQ chatbot

```text
Tao 15 cau hoi thuong gap cho chatbot ban quan ao: hoi theo ngan sach, danh muc, size, mau sac, giao hang, doi tra va cach dat hang.
```

## 9. Kiem thu chatbot

```text
Tao bang test chatbot gom cau hoi, ket qua mong muon, ket qua thuc te, dat/chua dat va ghi chu sua loi hallucination.
```

```text
Viet phan mo ta cong nghe cho bao cao do an: React/Vite cho frontend, N8N webhook cho chatbot workflow, du lieu san pham rieng va nguyen tac bao mat API key.
```

```text
Lap dan y slide thuyet trinh 7 slide cho do an website ban hang mini tich hop chatbot AI, gom van de, giai phap, tinh nang, kien truc, demo, kiem thu va ket luan.
```
