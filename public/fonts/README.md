# MiniStyle font files

Place licensed self-hosted webfont files here.

Expected filenames:

```text
GeneralSans-Regular.woff2
GeneralSans-Medium.woff2
GeneralSans-Semibold.woff2
EditorialNew-Regular.woff2
EditorialNew-Ultralight.woff2
```

`General Sans` is used for body/UI text. `Editorial New` is used for hero, page titles, and editorial headings.

Do not commit commercial font files unless the license allows storing them in this repository.
